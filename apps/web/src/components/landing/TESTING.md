# Landing Page Testing Checklist

## Visual Testing

### Desktop (1920x1080)
- [ ] Hero particles render and animate smoothly (60fps)
- [ ] Particles react to mouse movement
- [ ] Neural network floats and rotates
- [ ] Gradient orbs visible and animated
- [ ] Hero text is readable and centered
- [ ] CTA buttons hover states work
- [ ] Scroll indicator appears and bounces

### Feature Showcase
- [ ] Left side pins while scrolling
- [ ] Preview updates as features scroll into view
- [ ] Preview transitions are smooth
- [ ] Feature descriptions are readable
- [ ] Icons display correctly

### Stats Section
- [ ] Counters animate on scroll into view
- [ ] Final numbers are correct
- [ ] Animation is smooth (not jumpy)

### How It Works
- [ ] Steps pin correctly
- [ ] Progress indicator updates
- [ ] Active step highlights properly
- [ ] Icons glow on active step

### CTA Section
- [ ] Email form submits (mock)
- [ ] Success message displays
- [ ] Form resets after success
- [ ] Feature badges visible

### Navigation
- [ ] Navbar starts transparent
- [ ] Navbar becomes solid on scroll
- [ ] Section highlighting works
- [ ] Smooth scroll to sections
- [ ] Logo links to top

## Mobile Testing (375x667)
- [ ] 3D disabled, CSS fallback shows
- [ ] Gradient orbs animate
- [ ] Text is readable
- [ ] Buttons are tappable
- [ ] No horizontal scroll
- [ ] Mobile menu opens/closes
- [ ] Mobile menu links work
- [ ] Feature showcase doesn't pin (stacks)
- [ ] Stats display in grid
- [ ] How It Works stacks vertically

## Performance
- [ ] FPS stays above 55 (desktop)
- [ ] No jank on scroll
- [ ] Initial load < 3 seconds
- [ ] Time to Interactive < 5 seconds

## Accessibility
- [ ] Keyboard navigation works
- [ ] Focus indicators visible
- [ ] Reduced motion respected
- [ ] Color contrast meets WCAG AA
